# SmartStock MIS


## 📌 Overview
SmartStock MIS is an Inventory Management System developed using PHP and MySQL/MariaDB. It is designed to help users efficiently manage product stocks, monitor inventory levels, and organize item records in a structured and user-friendly way. The system aims to improve accuracy, reduce manual inventory work, and make stock tracking faster and more reliable for small to medium-sized businesses.

---

## ✨ Features
- ➕ Add new products/items  
- ✏️ Update product information  
- ❌ Delete items from inventory  
- 📊 Monitor stock levels  
- 🔍 Search and filter products  
- 💾 Database-driven storage (MySQL/MariaDB)  

---

## 🛠️ Tech Stack
- **Programming Language:** PHP  
- **Database:** MySQL / MariaDB  
- **Frontend:** HTML, CSS, JavaScript  
- **Server:** Apache (XAMPP / WAMP / Laragon)  
- **Environment:** Localhost development  

---

## 🚀 How to Run
1. Clone the repository: git clone https://github.com/maykolzxv/SmartStock-MIS.git
2. Move the project folder to: htdocs (if using XAMPP)
3. Start **Apache** and **MySQL** in XAMPP
4. Import the database:
    - Open phpMyAdmin
    - Create a database (e.g. `smartstock_db`)
    - Import the provided `.sql` file
5. Run the project in browser: http://localhost/SmartStock-MIS/
